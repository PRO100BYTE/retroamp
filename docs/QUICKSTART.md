# Quickstart

## Requirements

- Windows 10/11
- Node.js 18+

## Run in Development

```bash
npm install
npm run dev
```

## Build for Windows

```bash
npm run dist:win
```

Installer output is generated in `dist/`.

## Smoke Test

1. Start app.
2. Add a few files via [FILE].
3. Add folder via [FOLDER].
4. Try drag-and-drop from Explorer.
5. Verify playback, seek, volume, repeat/shuffle.
6. Verify keyboard shortcuts.
